# AGIAA-2K
A fine-grained dataset for Aesthetic and Alignment evaluation of AGIs. The paper has been accepted by ICASSP 2025. The AGIAA-2K dataset can be obtained through the Baidu Cloud Disk link:https://pan.baidu.com/s/1YKEbrZG65QJ_uzF9AHhsvg?pwd=AG1A Extraction code: AG1A
